# ビジネスプラン: DoggieDiet AI (愛犬向けパーソナライズ栄養管理アシスタント)

## 1. サービス概要
**DoggieDiet AI**は、**「データに基づいた精密な栄養管理」**をコアバリューとする、愛犬のためのパーソナライズ栄養管理ソリューションです。

---

## 2. コアバリュー (Value Proposition)

* **個別の最適化 (Personalization)**
    * 愛犬の犬種（ゴールデンレトリバー等）、年齢、体重、既往歴に基づき、一頭一頭に最適な栄養状態を分析します。
* **過不足の防止 (Nutritional Balance)**
    * 1日の摂取カロリーとPFCバランス（タンパク質・脂質・炭水化物）を可視化し、肥満や栄養不均衡を未然に防ぎます。
* **具体的な給餌ガイド (Actionable Guidance)**
    * その日の摂取データに基づき、「夕食はタンパク質中心のフードを150g追加してください」といった、飼い主がすぐに実行できる具体的なガイドを提供します。

---

## 3. 主要機能 (Key Features)

| 機能 | 内容 | 管理形式 |
| :--- | :--- | :--- |
| **プロフィール管理** | 基本情報（年齢・体重）および既往歴の保存 | Excel / CSV |
| **デイリー食事記録** | 摂取したフードの種類、量、成分の記録 | Excel / CSV |
| **AI栄養状態分析** | 蓄積されたデータを基にした現在の健康状態診断 | LLM (AI) |
| **本日の給餌処方** | 不足栄養素を補うための具体的な食事提案 | LLM (AI) |

---

## 4. 実装戦略 (Implementation Strategy)

Python(pandas)とLLMを組み合わせ、データ駆動型のアドバイスを自動生成するワークフローを構築します。

### 技術的アプローチ
1.  **データ管理 (Data Management)**
    * `pandas`ライブラリを使用して、エクセル形式のプロフィールデータと食事記録を読み込み、統合・整理します。
2.  **定量分析 (Quantitative Analysis)**
    * `pandas`で現時点までの総摂取カロリーおよびPFCバランスを計算し、目標値との乖離を数値化します。
3.  **定性分析および提案 (Qualitative Insights)**
    * 算出された数値データと愛犬の特性（疾患リスク等）をプロンプトとして**LLM**に伝達。専門家（獣医師・ペット栄養士）レベルのアドバイスを生成します。
